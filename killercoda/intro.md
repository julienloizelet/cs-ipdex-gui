# CrowdSec IPdex

Query IP threat intelligence from the CrowdSec CTI database.

## What This Does

IPdex allows you to:
- Look up threat intelligence for any IP address
- View reputation scores, behaviors, and classifications
- See geographic and network information
- Check if IPs are on known blocklists

## Prerequisites

Have this ready before clicking Start:

1. **CrowdSec CTI API Key** from [CrowdSec Console](https://app.crowdsec.net/cti-api-keys)

Click **Start** when ready.
